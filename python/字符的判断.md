# 字符在判断中的布尔值

空字符为False

非空字符为True

```python
if '':
	print('\'\' is viewed as True')
else:
    print('\'\' is viewed as False')
    
if 'a':
    print('\'a\' is view as True')
else:
    print('\'a\' is viewed as False')
```

输出：

```
'' is viewed as False
'a' is viewed as True
```

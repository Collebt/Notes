# import 和import * 的区别

1、` import numpy`

这是导入了整个numpy模块，需要使用句点表示法访问需要的类。例如：

`a=numpy.array([1,1])`

 

 

2、`from numpy import *`

这是导入了numpy模块的每个类，可以直接使用类，无需句点表示法。例如：

` a = array([1,1])`

 

不推荐使用第二种导入方式，其没有明确地指出你使用了模块中的哪些类。并且，如果导入了一个与程序文件中其他东西同名的类，会引发难以发现的错误。

 

如果需要直接导入所有def，可以考虑使用* 。

# Matlab矩阵索引（index）操作

想从矩阵A中取出坐标为(2,1),(3,2), 和 (4,4)的元素





如果直接使用A([2 3 4], [1 2 4])会取出索引覆盖到的所有元素，这个不符合目的。



### 在矩阵中取出离散元素

MATLAB默认矩阵中所有元素一个长**列向量**来存储数据

> 在python中是按照最后一维（通常为行）来储存序列

MATLAB提供一个函数：sub2ind，该函数能够将

特定行列下标转换为线性索引。

```
idx = sub2ind(size(A), [2 3 4], [1 2 4])
ans =
2 7 16
A(idx)
ans =
5 7  1
```
